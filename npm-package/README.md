
Stack Editor Server
----

Project https://github.com/Cirru/stack-editor

### Usage

```
stack-server
```

or

```
se
```

configure it with:

```bash
op=watch port=7010 extension=.cljs out=src stack-editor stack-sepal.ir
```

`op` for operation: `watch` or `compile`.

`stack-sepal.ir` is the default filename.
